# Tools

| Tool | Purpose |
|------|---------|
| cmr_collection_search | Search NASA CMR for Earth science dataset collections by keyword, variable, temporal range, and spatial extent |
| gcmd_keyword_lookup | Look up GCMD controlled vocabulary terms to normalize and expand search terms before querying CMR |
| semantic_scholar_search | Search scientific literature to refine variables, spatial/temporal scales, and processing levels for CMR search |
